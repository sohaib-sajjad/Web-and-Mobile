# Event Manager (Flutter)

Simple event management Flutter app with a Host side (create/manage events) and an Attendee side (browse/register).

Quick start

1. Ensure Flutter is installed and on your PATH: `flutter --version`
2. From the project folder run:

```bash
flutter pub get
flutter run
```
